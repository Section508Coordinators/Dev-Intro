Overview

Welcome to DHS OAST Section 508 Developer support site on the Section508Coordinators GitHub site!

This Section508Coordinators GitHub site has many useful areas and assets. To help orient those interested in the support available to developers and test automation engineers, DHS OAST has provided this series of GitHub navigational pages to serve as a roadmap to the technical support assets available to you on GitHub.

Who is this support targeted to?

The example code and other resources within the developer/tester support repositories are targeted to support the accessibility efforts of technical teams at the Department of Homeland Security (DHS).

What support areas are covered?

At this time, example support material fall into the-following categories:

  • Accessibility test tools
  • Integrating accessibility into test automation
  • Accessible technical authoring and developer coding tools
  • Developer and tester accessibility training resources
